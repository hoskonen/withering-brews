# Withering Brews

Withering Brews is a work-in-progress potion degradation mod for **Kingdom Come: Deliverance II**.

The goal is to make potions feel less like permanent, consequence-free inventory items. Potions gradually age in the background and will eventually lose quality.

## Concept

Kingdom Come: Deliverance II represents potion quality with separate items, usually ranging from quality I to IV.

Withering Brews uses those existing quality tiers:

```text
Quality IV → Quality III → Quality II → Quality I
